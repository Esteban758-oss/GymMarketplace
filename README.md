## Temática

Marketplace donde los usuarios tienen disponible una tienda en la cual se pueden comprar articulos para una vida fitness .


## Descripción

En este sitio tendremos a la venta distintas categorías de productos los cuales son un articulo tal como suplemento, prendas de gimnasio, elementos de gimnasio(pesas, soga, barra, etc)
